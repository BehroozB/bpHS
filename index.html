Getting Started

git clone https://github.com/boompx/bpHS.git

Markup

<div class="bp-hs">
  <div class="bp-hs_inner">
    <div class="bp-hs_inner__item">
      <img src="..." alt="..."/>
    </div>
  </div>
</div>

Usage

bpHS require the use of an id on the outermost container (.bp-hs) for slider controls to function properly.

Fire bpHS

$('.bp-hs').bpHS();

Options

The default effect/transition for bp-hs_inner__item is opacity. You can control the animation for each item via data attributes data-*.

Data attribute Description
data-transform="scale" Is used to scale elements. Add an addtional effect by using data-origin="top-(right | left)" or data-origin="bottom-(right | left)"
data-transform="rotate" Is used to rotate elements from corners based on data-origin="top-(right | left)" or data-origin="bottom-(right | left)" (required*)

Methods

Initializes the bpHS with an optional options

Name Type Default Description
activate Number 0 This event fires on bp-hs_inner__item and add is-active class to make visible/default.
touchSwipe Boolean true Allows you to drag slider items (right | left) on desktop and swipe on handheld devices. This option required TouchSwipe Jquery Plugin
nextText String null The text to display for the next control button
prevText String null The text to display for the previous control button
showControls Boolean true Show or hide all controls (next, prev and bullets)
showButtons Boolean true Show or hide next, prev only
showBullets Boolean true Show or hide bullets only
arrowKeys Boolean false Allows to navigate slider items on keypress arrow
autoPlay Boolean false Automatically cycling an item (paused on hover). If false, slider will not automatically cycle.
duration Number 5000 The amount of time to delay between automatically cycling an item.

more options are coming soon...
